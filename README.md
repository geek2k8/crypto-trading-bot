# crypto-trading-bot
## What is it?
- crypto market trading bot
- auto trading by the rule : buy low sell high
## How to run it?
- install nodejs
- install packages
- input your secret key, access key
- input your initial capital
- 
```javascript
node real_trade.js
```
- if you want test, node trade_virtual.js
